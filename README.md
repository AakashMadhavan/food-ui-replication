# Food UI Replication (React)

This project is a mobile UI screen replication of a food ordering application.

## Features

* Header with navigation
* Category tabs with active states
* Date selector
* Membership cards
* Food tickets (list layout)
* Combo deals (grid layout)
* Responsive design

## Tech Stack

* React
* Tailwind CSS
* Vite

## Run Locally

```bash
npm install
npm run dev
```
